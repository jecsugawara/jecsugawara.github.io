0. 前回の振り返り：おみくじを画像にする

echo で <img> タグを出力することで、ブラウザに画像を表示させます。

sample05.php

<?php
// 1か2をランダムに生成
$omikuji = rand(1, 2); 

if ($omikuji === 1) {
    echo "判定：大吉<br>";
    // <img>タグを出力。パスの指定に注意
    echo "<img src='picture/daikichi.png' width='150'>";
} else {
    echo "判定：小吉<br>";
    echo "<img src='picture/shokichi.png' width='150'>";
}
?>
【Javaエンジニアへのヒント】 echo の中で src='...' とシングルクォートを使っているのは、echo 全体を囲むダブルクォート（"）と衝突しないようにするためです。
width='150' のようにサイズを指定しておくと、画像が表示された時にレイアウトが崩れにくくなります。
📝 演習 ex05.php 【問題】乱数の範囲を 1〜3 に変更してください。「3」が出た場合には 「中吉（chukichi.png）」 が表示されるように、elseif を使ってプログラムを拡張してください。

1. 配列と連想配列の基本

PHPの配列は、Javaの ArrayList と HashMap の両方の性質を持っています。

sample06_01.php（インデックス配列）

<?php
$ary = ["a", "b", "c"];
$ary[] = "d"; // 末尾に追加
echo $ary[0]; // 0から始まる
?>

sample06_02.php（連想配列）

<?php
$user = ["name" => "田中", "age"  => 25]; // キー => 値
echo $user["name"];   // キーを指定して表示
?>
📝 演習 ex06.php 【問題】自分の「好きな食べ物」を3つ入れた配列を作成し、var_dump で中身（型や要素数）を確認してください。

2. 配列のループ処理 (foreach)

配列の全要素を表示する場合、PHPでは foreach を使うのが一般的です。

sample07.php

<?php
$colors = ["red" => "赤", "blue" => "青"];
foreach ($colors as $key => $value) { // キーと値を取り出す
    echo "$key は日本語で $value です。<br>";
}
?>
📝 演習 ex07.php 【問題】自分の「氏名」「年齢」「出身地」をキーにした連想配列を作成し、foreach を使って「項目名：値」の形式ですべて表示してください。

3. データの送信（POST/GET）とテキストボックス

method 属性によってデータの受信方法が変わります。入力フォームは拡張子 .html で作成します。

sample08.html

<!DOCTYPE html>
<html lang="ja">
<head><meta charset="UTF-8"><title>POST送信</title></head>
<body>
    <form method="post" action="sample08.php">
        名前: <input type="text" name="txt1">
        <input type="submit" value="送信">
    </form>
</body>
</html>

sample08.php

<!DOCTYPE html>
<html lang="ja">
<head><meta charset="UTF-8"><title>受信</title></head>
<body>
<?php echo "入力された内容: " . $_POST['txt1']; ?>
</body>
</html>

4. ラジオボタンの受信

ラジオボタンは選択された1つの値が送信されます。

sample09.html / sample09.php

<!DOCTYPE html>
<html lang="ja">
<head><meta charset="UTF-8"><title>ラジオボタン</title></head>
<body>
    <form method="post" action="sample09.php">
        性別: 
        <input type="radio" name="gender" value="1">男
        <input type="radio" name="gender" value="2">女
        <input type="submit" value="送信">
    </form>
</body>
</html>
<?php
$sei = isset($_POST['gender']) ? $_POST['gender'] : 3;
switch ($sei) {
    case 1: echo "男性ですね"; break;
    case 2: echo "女性ですね"; break;
    default: echo "未選択です"; break;
}
?>

5. チェックボックス（配列形式）

複数選択させる場合は、name 属性に [] を付けて配列として受け取ります。

sample10.html / sample10.php

<!DOCTYPE html>
<html lang="ja">
<head><meta charset="UTF-8"><title>チェックボックス</title></head>
<body>
    <form method="post" action="sample10.php">
        趣味:
        <input type="checkbox" name="hobby[]" value="音楽">音楽
        <input type="checkbox" name="hobby[]" value="映画">映画
        <input type="submit" value="送信">
    </form>
</body>
</html>
<?php // sample10.php
if (isset($_POST['hobby'])) {
    foreach ($_POST['hobby'] as $value) { 
        echo "趣味: $value <br>";
    }
}
?>
📝 演習 ex10.html / ex10.php 【問題】「使用経験のあるプログラミング言語」を複数回答できるチェックボックス（Java, PHP, Pythonなど）を作成し、受信側で選んだものをすべて表示してください。

ex10.html

<body>

</form>
</body>
</html>

6. エラーチェック（empty / isset）

入力値の状態によって関数を使い分けます。

sample11.html/sample11.php

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>入力チェック用フォーム</title>
</head>
<body>
    <form method="post" action="sample11.php">
        名前：<input type="text" name="txt1"><br>
        性別：
        <input type="radio" name="gender" value="男">男
        <input type="radio" name="gender" value="女">女
        <input type="submit" value="送信">
    </form>
</body>
</html>
<?php
// テキストボックスが空（未入力）かチェック
if (empty($_POST['txt1'])) {
    echo "名前を入力してください";
}

// ラジオボタン等がセットされているかチェック
if (!isset($_POST['gender'])) {
    echo "性別を選択してください";
}
?>

7. 三項演算子

if ～ else を 1 行で書く方法です。

sample12.html/sample12.php

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>名前入力フォーム</title>
</head>
<body>
    <form method="post" action="sample11.php">
        名前：<input type="text" name="name">
        <input type="submit" value="送信">
    </form>
</body>
</html>
<?php
// 変数 = 条件 ? 真の場合 : 偽の場合;
$msg = empty($_POST['name']) ? "名無しさん" : $_POST['name'];
echo "こんにちは、" . $msg . "さん";
?>
