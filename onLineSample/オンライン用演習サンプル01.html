第1回：PHPの基本文法と制御構造

1. PHPの基本文法

PHPは変数の前に $ を付けます。文字列の連結には + ではなく .（ドット） を使用するのがJavaとの大きな違いです。

sample01.php

<?php
// コメント：Javaと同じ
$subject = "Webプログラミング";

// 変数展開（ダブルクォートは変数の中身を表示できる）
echo "Hello $subject\n";

// 文字列連結（ドットに注意！）
echo 'Hello ' . $subject; 

// デバッグ（型と長さを表示。JavaのSystem.out.printlnより詳細）
var_dump($subject); 
?>
📝 演習 ex01.php 自分の学籍番号と氏名を別々の変数に代入し、それらを連結してフルネームで表示してください。
また、その結果を var_dump で出力して、文字列の長さ（バイト数）がどうなっているか確認しましょう。

2. ループ（繰り返し）

for 文の構造はJavaと同じです。PHPではHTMLタグを echo することで動的なページを作成します。

sample02.php

<?php
for ($i = 1; $i <= 5; $i++) {
    // HTMLタグもただの文字列として出力できる
    echo "<li>項目 $i</li>";
}
?>
📝 演習 ex02.php カウンター変数 $i を 10 から始まり 1 で終わる ように変更し、カウントダウンを表示させてください。

3. 条件分岐：if, else, elseif

乱数を発生させる rand(最小, 最大) 関数を使います。Javaの Random クラスより手軽に扱えます。

sample03.php

<?php
$omikuji = rand(1, 2); // 1か2をランダムに生成

if ($omikuji === 1) {
    echo "大吉";
} else {
    echo "小吉";
}
?>
📝 演習 ex03.php 乱数の範囲を 1〜3 に広げ、elseif を追加して「中吉」も表示されるようにしてください。

4. 条件分岐：switch-case

選択肢が多い場合は switch 文が見やすくなります。Java同様、break; を忘れないようにしましょう。

sample04.php

<?php
$omikuji = rand(1, 3);

switch ($omikuji) {
    case 1:
        echo "大吉";
        break;
    case 2:
        echo "中吉";
        break;
    case 3:
        echo "小吉";
        break;
}
?>
📝 演習 ex04.php 上記の switch 文に default: を追加してください。もし変数 $omikuji が 1〜3 以外だった場合に、「末吉」 を表示するように改造してください。
