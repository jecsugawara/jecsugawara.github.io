0_1. 振り返り：セレクトボックス入力と表示

【プログラム説明】
HTMLの <select> を使い、ドロップダウンリストからデータを選択して送信します。PHP側では、選択された option の value を $_POST で取得します。

sample13.html

<!DOCTYPE html>
<html lang="ja">
<head><meta charset="UTF-8"><title>セレクトボックス送信</title></head>
<body>
    <form method="post" action="sample13.php">
        <p>好きなOSを選択：</p>
        <select name="os">
            <option value="Windows">Windows</option>
            <option value="macOS">macOS</option>
            <option value="Linux">Linux</option>
        </select>
        <input type="submit" value="送信">
    </form>
</body>
</html>

sample13.php

<?php
$os = $_POST['os'];
echo "あなたが選んだOSは " . $os . " です。";
?>

0_2. 振り返り：リンクでデータを送る(GETの応用）

【プログラム説明】
<a href="URL?キー=値"> という形式でリンクを作成すると、送信ボタンを使わずにデータを送ることができます。受け取り側のPHPでは $_GET を使って値を取得します。今回は数値（1〜3）に応じて表示する画像を切り替えます。

sample14.html

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>sample14</title>
</head>
<body>
    <h1>今日の天気を選んでください</h1>
    <ul>
        <li><a href="sample14.php?weather=1">晴れ</a></li>
        <li><a href="sample14.php?weather=2">曇り</a></li>
        <li><a href="sample14.php?weather=3">雨</a></li>
    </ul>
</body>
</html>

sample14.php

<?php
$weather = $_GET['weather'];

if ($weather == "1") {
    echo '<img src="picture/sunny.png" alt="晴れ">';
} elseif ($weather == "2") {
    echo '<img src="picture/cloudy.png" alt="曇り">';
} elseif ($weather == "3") {
    echo '<img src="picture/rainy.png" alt="雨">';
}
?>

1. 関数の基本（引数のない関数）

【プログラム説明】
決まった処理をまとめて名前をつけることで、どこからでも呼び出せるようになります。まずは引数（データ）を渡さないシンプルな関数を定義します。

sample15.html

<!DOCTYPE html>
<html lang="ja">
<head><meta charset="UTF-8"><title>引数なし関数送信</title></head>
<body>
    <form method="post" action="sample15.php">
        <input type="submit" value="メッセージ実行">
    </form>
</body>
</html>

sample15.php

<?php
// 関数の定義
function foo() {
    echo "Hello PHP!";
}

// 関数の呼び出し
foo();
?>

2. 引数のある関数

【プログラム説明】
関数に外からデータ（引数）を渡して、その値に基づいた処理を行います。PHPは基本が「値渡し」なので、関数内で値を書き換えても元の変数には影響しません。

sample16.html

<!DOCTYPE html>
<html lang="ja">
<head><meta charset="UTF-8"><title>引数テスト</title></head>
<body>
    <form method="post" action="sample16.php">
        数値を入力：<input type="text" name="val">
        <input type="submit" value="計算実行">
    </form>
</body>
</html>

sample16.php

<?php
function bar($var1) {
    $var1 = $var1 * 2;
    echo "関数内の計算結果：" . $var1 . "<br>";
}

$a = $_POST['val'];
bar($a); 
echo "関数呼び出し後の元の変数の値：" . $a;
?>

3. 戻り値のある関数

【プログラム説明】
関数で計算した結果を、return 文を使って呼び出し元に返します。戻された値は変数に代入して再利用できます。

sample17.html

<!DOCTYPE html>
<html lang="ja">
<head><meta charset="UTF-8"><title>戻り値テスト</title></head>
<body>
    <form method="post" action="sample17.php">
        数値1：<input type="text" name="n1"><br>
        数値2：<input type="text" name="n2"><br>
        <input type="submit" value="足し算">
    </form>
</body>
</html>

sample17.php

<?php
function add($var1, $var2) {
    return $var1 + $var2;
}

$a = $_POST['n1'];
$b = $_POST['n2'];

$ans = add($a, $b); 
echo "計算結果：" . $ans;
?>

4. include と require

【プログラム説明】
他のファイルに記述されたPHPプログラムを読み込んで実行します。共通の関数集などを別ファイル（ライブラリ）として管理する際に利用します。include は読み込みに失敗しても警告のみで処理を続行しますが、require はエラーとなり処理を停止します。

【呼ばれる側】lib.php

<?php
    echo "lib.php:外部ライブラリを読み込みました。<br>";
?>

【呼ぶ側】sample18.php

<?php
include("lib.php"); // ファイルの読み込み

echo "sample18.php";
?>

5. require_once

【プログラム説明】
ファイルを「一度だけ」読み込みます。複数のファイルで共通ライブラリを読み込んでいる場合、重複読み込みによるエラーを防ぎます。

sample19.php

<?php
// 同じファイルを2回指定しても、1度しか読み込まれない
require_once("lib.php");
require_once("lib.php");

echo "sample19.php";
?>
💡 セクションのポイント _once 系の関数は、読み込み済みのファイルを記憶しているため、二重定義によるシステムダウンを防止できます。

第3回 振り返りプリント

  • 関数： function で定義し、名前で呼び出す。
  • 引数： 関数に渡す値。デフォルトは「値渡し」。
  • 戻り値： return で結果を返す。
  • 読み込み： require_once は開発現場で最も多用される読み込み方法。
