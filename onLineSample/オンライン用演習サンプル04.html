1. DBの接続

【プログラム説明】
PDOを使用してDBに接続します。接続用ファイルを独立させることで、各プログラムからの接続処理を簡略化します。

db_connect.php

<?php
require_once("config.php");
$dsn = 'mysql:host=' . DB_HOST . ';dbname=' . DB_NAME . ';charset=utf8';
try {
    $pdo = new PDO($dsn, DB_USER, DB_PASS);
} catch (PDOException $e) {
    exit('データベース接続失敗。' . $e->getMessage());
}
?>

sample22.php

<?php
require_once("db_connect.php");
echo "データベース接続に成功しました。";
?>

2. データの表示（1件）

【プログラム説明】
特定の条件に合致するデータを1件取得します。SQL文では「*」を使わず、取得したい列名を明確に指定します。

sample23.php

<?php
require_once("db_connect.php");

// 列名を明記して取得
$sql = "SELECT id, username, email FROM w_user WHERE id = 1";
$stmt = $pdo->prepare($sql);
$stmt->execute();

$row = $stmt->fetch(PDO::FETCH_ASSOC);
if ($row) {
    echo "ID: " . $row['id'] . " 氏名: " . $row['username'];
}
?>
📝 演習 ex23.php

w_user テーブルからIDが2のレコードの「username」と「address」を列名指定で取得し、表示するプログラムを作成しなさい。

💡 セクションのポイント 列名を明記することで、必要なデータのみを取得し、セキュリティとパフォーマンスを向上させます。

3. データの表示（全件）

【プログラム説明】
テーブル内のすべてのデータを列名指定で取得し、繰り返し処理で表示します。

sample24.php

<?php
require_once("db_connect.php");

$sql = "SELECT id, username FROM w_user";
$stmt = $pdo->prepare($sql);
$stmt->execute();

while ($row = $stmt->fetch(PDO::FETCH_ASSOC)) {
    echo $row['id'] . "：" . $row['username'] . "<br>";
}
?>

4. プレースホルダーを使った検索

【プログラム説明】
入力画面から送られた値を使って検索します。SQLインジェクション対策としてプレースホルダーを使用します。
検索結果がない場合の処理を含め、安全に検索機能を実装します。

sample25.html（入力画面）

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>検索入力</title>
</head>
<body>
    <form method="post" action="sample25.php">
        氏名検索: <input type="text" name="u_name">
        <input type="submit" value="検索">
    </form>
</body>
</html>

sample25.php（検索結果表示）

<?php
require_once("db_connect.php");

$name = $_POST['u_name'];
$sql = "SELECT username, address FROM w_user WHERE username LIKE :target";
$stmt = $pdo->prepare($sql);
$stmt->bindValue(':target', '%' . $name . '%');
$stmt->execute();

$results = $stmt->fetchAll(PDO::FETCH_ASSOC);

if (empty($results)) {
    echo "検索結果は見つかりませんでした。";
} else {
    foreach ($results as $row) {
        echo $row['username'] . "（" . $row['address'] . "）<br>";
    }
}
?>

5. リンクを使った検索

【プログラム説明】
リンクから渡されるパラメータを安全に取得・表示します。

sample26.html（選択画面）

<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>ユーザー選択</title>
</head>
<body>
    <ul>
        <li><a href="sample26.php?uid=1">ID:1の詳細</a></li>
        <li><a href="sample26.php?uid=3">ID:3の詳細</a></li>
    </ul>
</body>
</html>

sample26.php（詳細表示）

<?php
require_once("db_connect.php");
$id = $_GET['uid'];
$sql = "SELECT username, email FROM w_user WHERE id = :id";
$stmt = $pdo->prepare($sql);
$stmt->bindValue(':id', $id, PDO::PARAM_INT);
$stmt->execute();
$row = $stmt->fetch(PDO::FETCH_ASSOC);
if ($row) {
    echo "名前：" . $row['username'] . "<br>";
    echo "Email：" . $row['email'];
} else {
    echo "指定されたユーザーは見つかりません。";
}
?>

第4回 振り返りポイント

  • config.php： 接続設定を分離し、require_onceで読み込む。
  • 列名の指定： SELECT * ではなく必要な列名を明記する。
  • 結果の判定： データがない場合のメッセージ処理もプログラムの品質として重要。
